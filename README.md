# Excalidraw-Whiteboard
https://excalidraw.com/


Excalidraw 是一款非常受歡迎的虛擬白板工具，以其獨特的「手繪風格」（Hand-drawn style）而聞名。它由 Zsviczian（Zsolt Viczian）等人開發與維護，是一個開源項目。

如果您看到的名稱是 "Excalidraw-Whiteboard"，這通常是指它在筆記軟體 Obsidian 中的插件名稱，或者是 VS Code 中的擴充套件。

以下是關於 Excalidraw 的詳細介紹與核心特色：
1. 核心特色 (Core Features)

Excalidraw 與傳統的 Visio 或 Draw.io 不同，它的設計理念不是「精確」，而是「快速表達想法」。

    手繪質感 (Sketchy Look)： 線條看起來像是用筆手繪的，這讓圖表看起來不那麼正式，非常適合用於草稿、構思階段 (Brainstorming) 或原型設計。這種風格降低了「必須畫得很完美」的心理壓力。

    無限畫布 (Infinite Canvas)： 沒有邊界，您可以隨意縮放和平移，適合繪製龐大的系統架構圖或心智圖。

    隱私與加密 (End-to-End Encryption)： 官方網頁版支援端到端加密協作，伺服器端無法讀取您的畫作內容。

    豐富的元件庫 (Libraries)： 社群提供了大量的圖示庫，包含 AWS/Azure 架構圖標、軟體工程符號、電路圖等，可以直接拖拉使用。

    輕量化與開源： 基於 React 開發，載入速度極快，且沒有複雜的選單干擾。

2. 主要應用場景 (Use Cases)

    系統架構設計 (System Design)： 工程師常用它來畫後端架構、資料流向圖。

    演算法與流程圖： 解釋複雜邏輯時，手繪風格的箭頭和方塊非常直觀。

    線框圖 (Wireframing)： UI/UX 設計師用來快速繪製產品介面草圖。

    教學與簡報： 許多技術部落格或簡報喜歡使用 Excalidraw 的風格，因為它比制式圖表更具親和力。

3. 生態系與整合 (Integrations)

這是 Excalidraw 最強大的地方，它不僅僅是一個網站，還能嵌入到開發者的工作流中：
平台	說明
Web (excalidraw.com)	最基礎的版本，打開瀏覽器即可用，支援即時多人協作。
Obsidian (Plugin)	最強大的整合。 允許在 Obsidian 筆記中直接嵌入可編輯的 Excalidraw 畫布，甚至可以將筆記之間的「雙向連結」視覺化。支援 OCR (圖片轉文字) 和腳本自動化。
VS Code (Extension)	允許開發者在程式碼編輯器中直接畫架構圖，檔案通常以 .excalidraw 或 .excalidraw.json 儲存，與程式碼一起版本控制 (Git)。
4. 檔案格式與匯出

    原始檔： 儲存為 JSON 格式（通常副檔名為 .excalidraw）。

    匯出： 支援匯出為 PNG、SVG。

    深色模式 (Dark Mode)： 完美支援深色主題，匯出時也可選擇是否保留深色背景。

5. 為什麼選擇 Excalidraw？

如果您需要畫正式的工程藍圖（如建築圖、精確尺寸的電路板），AutoCAD 或 Draw.io 可能更適合。

但如果您需要：

    快速溝通想法。

    畫出有溫度、不呆板的架構圖。

    將圖表與您的筆記 (Obsidian) 或程式碼 (VS Code) 緊密結合。

那麼 Excalidraw 是目前的最佳選擇。
